## rest-api-testing

### Description
rest-api-testing is a simple daemon used to test REST API calls from the client.

### Installation
```shell
git clone https://github.com/mskrha/rest-api-testing.git
cd rest-api-testing
make
```

### Usage
```shell
./rest-api-testing
```
